# Stock Portfolio Tracker

## Description :
A simple tool that allows users to track their stock investments. It calculates the total value of a portfolio based on hardcoded real-time prices and user-defined quantities.

## Features :
* User input for stock symbols and quantity.
* Validation against a predefined dictionary of stock prices.
* Calculation of individual stock value and total portfolio value.
* **File Export:** Automatically saves the final report to `portfolio.txt`.
